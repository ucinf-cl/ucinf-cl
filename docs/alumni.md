# Alumni UCINF

![Logo UCINF](assets/images/ucinf_logo.jpeg){ .small-logo }

Esta seccion esta pensada para exalumnos que quieran mantenerse conectados con la historia de UCINF y con su comunidad.

## Que puedes encontrar aqui

- Informacion base sobre el cierre institucional.
- Un canal de contacto para iniciativas de comunidad.
- Espacio para compartir recuerdos, testimonios y actividades entre exalumnos.

## Informacion relevante

- UCINF finalizo actividades el **31 de diciembre de 2019**.
- El cierre implico el termino de la atencion presencial en la sede de Ricardo Lyon.
- Este sitio no es continuador academico ni administrativo de la universidad.

## Comunidad alumni

La red alumni busca fortalecer el vinculo entre generaciones, facilitar reencuentros y mantener vivo el legado de UCINF en la vida profesional y personal de sus exalumnos.

## Contacto

Para consultas o coordinacion de actividades: **alumni@ucinf.cl**.
